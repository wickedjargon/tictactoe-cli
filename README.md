# Tic-Tac-Toe in Rust

A simple console-based Tic-Tac-Toe game written in Rust.

## Running the Game

1. Clone the repository:
    ```bash
    git clone git@github.com:wickedjargon/tictactoe-cli.git
    ```
2. Navigate to the project directory:
    ```bash
    cd tictactoe-cli
    ```
3. Run the game:
    ```bash
    cargo run
    ```
